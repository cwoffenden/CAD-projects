# ZuluSCSI Wide SCA bracket

Alternative 3D printed 3.5" mounting bracket for a [ZuluSCSI Wide SCA](//shop.rabbitholecomputing.com/products/zuluscsi-wide-sca).

![3D printed bracket](doc/bracket.jpeg)

This was originally designed for an SGI Octane sled (which flexes with the original part) but should fit any other expecting a standard drive.

![SGI Origin sled with ZuluSCSI Wide SCA](doc/origin-sled.jpeg)
